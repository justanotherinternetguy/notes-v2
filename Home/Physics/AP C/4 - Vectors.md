# Vector Resolution
- Split a vector $\vec{V}$ into X and Y components

$$\vec{V} = \vec{V}_x + \vec{V}_y$$
$$V_x = V \cos \theta$$
$$V_y = V \sin \theta$$
$$V = \sqrt{{V_x}^2 + {V_y}^2}$$
$$\tan \theta = \frac{V_y}{V_x}$$
$$\theta = \tan^{-1} \frac{V_y}{V_x}$$
