# Equation Notation
## Prefix
- Each operator is placed **before** operands
- $5 + \frac{8}{3-1}$ $\rightarrow$ `+ 5 / 8 - 3 1` (prefix)
- $\rightarrow$ `5 8 3 1 - / +` (postfix)
